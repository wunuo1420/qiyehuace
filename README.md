# 企业画册

一个支持翻页效果的企业画册网页，支持单页展示、滑动和点击翻页。

## 功能特点

- 单页展示，每次翻一页
- 支持点击翻页
- 支持触摸滑动翻页（移动端）
- 支持键盘翻页（右箭头/空格键）
- 响应式设计，适配各种屏幕尺寸
- 优化微信分享体验

## 本地运行

使用Python启动本地服务器：

```bash
python -m http.server 8000
```

然后在浏览器中访问：http://localhost:8000

## 部署

### GitHub Pages

1. 在GitHub上创建新仓库
2. 推送代码到GitHub
3. 在仓库设置中启用GitHub Pages

详细步骤请参考：[GitHub Pages官方文档](https://docs.github.com/en/pages)
